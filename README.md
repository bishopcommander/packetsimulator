# 🚀 Network Packet Simulator

A visual and interactive **Network Packet Simulator** built using **Spring Boot + JavaScript (vis.js)** that demonstrates routing algorithms like **BFS and Dijkstra** with real-time packet animation and delay analysis.

---

## 🎯 Features

- 🔀 **Algorithm Toggle**
  - BFS (Shortest path by hops)
  - Dijkstra (Shortest path by weight)

- 📦 **Packet Simulation**
  - Step-by-step packet movement
  - Smooth animation across nodes
  - Edge highlighting during traversal

- ⚖️ **Weighted Graph Support**
  - Assign weights to edges
  - Visual display of weights

- 📊 **Metrics Panel**
  - Hop count
  - Total delay (based on edge weights)

- ⚠️ **Packet Loss Simulation**
  - Configurable packet loss probability

- 🧾 **Structured Logging**
  - Clean console logs for debugging and analysis

- 🎨 **Modern UI**
  - Dark theme dashboard
  - Interactive graph controls

---

## 🧠 Concepts Demonstrated

- Graph Data Structures
- Breadth-First Search (BFS)
- Dijkstra’s Shortest Path Algorithm
- Network Routing Simulation
- Packet Transmission & Loss

---

## 🛠️ Tech Stack

- **Backend:** Spring Boot (Java)
- **Frontend:** HTML, CSS, JavaScript
- **Visualization:** vis.js (network graph)
